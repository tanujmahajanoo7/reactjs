# Background Color Changer

A simple React application that allows users to change the background color by clicking buttons.

## 📋 Description

This project demonstrates:
- useState hook for managing component state
- Event handling in React
- Dynamic className updates based on state
- Tailwind CSS for styling

## 🎯 Features

- Interactive background color changing
- Six color options: Red, Blue, Green, Cyan, Sky, and Indigo
- Smooth UI with Tailwind CSS
- Responsive design

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm

### Installation

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

The app will open at `http://localhost:5173` (or another port if 5173 is busy).

## 📦 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build

## 🏗️ Project Structure

```
src/
  ├── App.jsx       # Main component with color changer logic
  ├── main.jsx      # Application entry point
  └── index.css     # Global styles with Tailwind directives
```

## 💡 Key Concepts

- **useState Hook**: Managing color state
- **Event Handlers**: onClick handlers for buttons
- **Conditional Styling**: Dynamic className based on state
- **Tailwind CSS**: Utility-first styling

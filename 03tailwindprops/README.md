# Tailwind CSS Props Example

A React application demonstrating the use of props in React components styled with Tailwind CSS.

## 📋 Description

This project showcases how to:
- Pass props to React components
- Use Tailwind CSS for styling
- Create reusable components with customizable props
- Handle default props in React

## 🎯 Features

- Reusable Card component
- Props passing (username, post, arrays, objects)
- Tailwind CSS styling
- Default prop values

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm

### Installation

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

The app will open at `http://localhost:5173` (or another port if 5173 is busy).

## 📦 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build

## 💡 Key Concepts

- **Props**: Passing data from parent to child components
- **Tailwind CSS**: Utility-first CSS framework
- **Component Reusability**: Creating components that accept different props

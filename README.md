# React Projects Collection

This repository contains a collection of React projects demonstrating various concepts, from basic React setup to advanced state management and routing.

## 📁 Project Structure

### Basic React Projects
- **01basicreact** - Basic React application using Create React App
- **01basicvite** - Basic React application using Vite

### Styling Projects
- **03tailwindprops** - React components with Tailwind CSS and props
- **04bgChanger** - Background color changer application

### Utility Applications
- **05passwordGenerator** - Password generator with customizable options
- **06currencyConverter** - Currency converter application

### Routing & State Management
- **07reactRouter** - React Router implementation with multiple pages
- **08miniContextapi** - Context API implementation example
- **09themeswitcher** - Theme switcher using Context API
- **10todoContextLocal** - Todo application with Context API and local storage
- **11reduxToolKit** - Todo application using Redux Toolkit

### React Hooks
- **Hook** - Collection of React Hooks examples
  - **01_UseState** - useState hook examples

## 🚀 Getting Started

Each project is independent and can be run separately. Navigate to any project folder and follow its specific README instructions.

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation
```bash
# Navigate to any project folder
cd <project-folder>

# Install dependencies
npm install

# Run the development server
npm run dev
# or
npm start
```

## 📚 Learning Path

1. Start with **01basicreact** or **01basicvite** to understand React fundamentals
2. Move to **03tailwindprops** to learn about props and Tailwind CSS
3. Practice with utility apps like **04bgChanger**, **05passwordGenerator**, and **06currencyConverter**
4. Learn routing with **07reactRouter**
5. Explore state management with **08miniContextapi**, **09themeswitcher**, and **10todoContextLocal**
6. Advance to **11reduxToolKit** for Redux state management
7. Study React Hooks in the **Hook** folder

## 🛠️ Technologies Used

- React

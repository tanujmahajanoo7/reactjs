# Basic React with Vite

A basic React application setup using Vite as the build tool.

## 📋 Description

This project demonstrates the initial setup of a React application using Vite. It includes a simple component structure with a YouTube component example.

## 🎯 Features

- React 18 with Vite
- Fast Hot Module Replacement (HMR)
- ESLint configuration
- Basic component structure

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm

### Installation

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

The app will open at `http://localhost:5173` (or another port if 5173 is busy).

## 📦 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build

## 🏗️ Project Structure

```
src/
  ├── App.jsx        # Main App component
  ├── main.jsx       # Application entry point
  └── Youtube.jsx    # YouTube component example
```

## 💡 Key Concepts

- **Vite**: Fast build tool and dev server
- **React Components**: Basic component structure
- **HMR**: Hot Module Replacement for fast development
